BinaryToString
==============

#Usando o BinaryToString:

##Cria-se a instancia da 'classe'
>var bs = new BinaryToString();

##Convertendo String para binario:
>bs.toBin('A')
//Retorna 00010100;

##Convertendo Binario para String:
>bs.binToString(00010100);
//Retorna 'A'
